

   Структура и компоненты программы на языке С++

   В Visual Studio программа представлена в виде иерархии объектов.

1. Решение (рабочая область) – совокупность нескольких проектов. Файл имеет 
расширение Имя.sln. Решение может содержать несколько проектов.

2. Проект - объединение нескольких модулей программного кода для совместной 
компиляции и сборки. Файл имеет расширение  Имя.vcproj,  файл проекта 
содержит описание модулей проекта и опций проекта.
Программа входит в состав проекта в виде модулей.

3. Модуль - файл, содержащий алгоритмы и данные для их обработки.
Программа на языке С++ состоит из одного или нескольких модулей - 
текстовых файлов с расширением "сpp" (Source).
Заголовочные файлы (с расширениями "h" или "hpp" - Header) 
могут входить в проект как включаемые файлы.


Однофайловые программы

Программа (модуль) может состоять из одной или нескольких функций.
Одна из функций обязательно должна иметь имя "main". С этой функции 
начинается выполнение программы.

void main(void)
{
}

Имя главной функции проекта main.

В одном файле может быть одна или несколько функций. 
Все функции С++ равноправны и описываются на одном уровне, но main главнее.

Любая функция, кроме main(), вызывается из другой функции.

При вызове функции ей могут быть переданы параметры (данные).

По окончании выполнения функции в вызывающую функцию может быть 
возвращено значение, а может не быть возвращено ничего (результат).

Пример обращения к библиотечным функциям.  

у=sin(x)   а=fabs(a);     //  Возвращают значение.
printf("x=%f y=%f",x,y);  //  Не возвращает значения, реализуя интерфейс.

Функция обладает типом, соответствующим типу возвращаемого значения.

1. Если функция ничего не возвращает, ее тип void.
void main (int argc, char* argv[])
Параметры передаются при запуске приложения из ОС.
void My_Print(void)
{
   for (int i=1;i<50;i++)
      printf('*');
   printf('\n');
}

2. Если тип функции опущен, то по умолчанию он будет int.
main (void)           // тип функции по умолчанию будет int
{

   return 1;
}

3. Если функция не имеет аргументов, вместо них в списке 
параметров записывается слово void.
void main (void)      //Не имеет типа и не имеет параметров
   {
   }

4. Если список параметров опущен, то аргументов может быть произвольное число
   void main() 		 //Параметров может быть много  

printf("%d%d%d",a,b,c);

int max()
   {
       ...
       return что-то;
   }


   Замечания.
   В С++ объявление переменой (объекта) возможно не только в начале программы, но и в любом месте кода до первого обращения.
   for (int i=0; i<10; i++)
     {
       a[i]     
     }  

   Область действия - только охватывающий блок {}.
Как правило, так объявляют рабочие переменные.

   БЛОК - любая последовательность операторов в { тело блока }
Назначение блока: структурирование кода.

Структура блока:
	{
	   // объвления переменных
           тело блока;
	}


Правила записи кода.
   ;       - заканчивает каждый оператор;

   {блок}  - размещается в любом месте программы;

   структура программы подчеркивается отступами (опция Indent)

   Комментарии - умеренное число необходимо.

   Имена объектов подчеркивают их смысл.
